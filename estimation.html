Estimation Session

Small Council Assembly

  1. Small council meetup to discuss issues that are new or parked
  2. Then identify all possible blockers
  3. Write a simple specification for suggested solution
  4. Mark JIRA as ready for estimation

Estimate Session

  1. To be done right after Small Council Assembly, everyone in teams should participate
  2. Small council to present each JIRA to be estimated
  3. JIRA description and suggested solution is quickly explained
  4. If one or more questions are not answered, JIRA should be parked and to be re evaluated on the next Small Council Assemply
  5. Estimation will be done online using https://www.pointingpoker.com/
  6. The average of the result will be used as the Story Point for the JIRA.
